# Repository Name
JakeFrosty-Cogs

# Installation
To install these cogs, run these commands ([p] is your bot prefix):
```
[p]repo add jakefrosty-cogs https://github.com/JakeFrosty/JakeFrosty-Cogs
```
```
[p]cog install jakefrosty-cogs <cog_name>
```
